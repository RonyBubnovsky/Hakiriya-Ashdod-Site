# Hakiriya Ashdod - School Website

## About

This project is a website I built for a school called "Hakiriya" in Ashdod, Israel.
I developed it as part of my work as a programming instructor at LeadersApp.

The school did not have a website and they needed one, so I created this solution for them.

## Live Website

The site is hosted on Vercel and can be viewed at: [https://hakiriya-ashdod.vercel.app](https://hakiriya-ashdod.vercel.app/)

## Pages

The website includes the following pages:

- **Home** - The main landing page with general information about the school
- **Mivne Irguni** - The organizational structure of the school
- **Tfisa Hinukhit** - The educational approach and philosophy
- **Yeadim** - Goals and objectives
- **Yozamot** - Initiatives and projects

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion (for animations)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/RonyBubnovsky/Hakiriya-Ashdod-Site.git
```

2. Install the dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Build

To create a production build:

```bash
npm run build
npm start
```
